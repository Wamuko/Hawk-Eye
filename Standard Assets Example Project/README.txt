ダウンロードしていただきありがとうございます。

本プロジェクトは、飛行機を操作して60秒以内にリングをどれだけくぐれるか？

という趣旨のゲームになっております。

起動方法：
このリポジトリをダウンロードして、Hawk Eye.exeをクリックしてください。

操作方法：

w s : 上下運動
a d : 旋回運動
右クリック：ブレーキ

リスタート：左上のボタンをクリック

このプロジェクトはUnity 2017.3.0f3を用いて作られました。
