ダウンロードしていただきありがとうございます。

本プロジェクトは、飛行機を操作して60秒以内にリングをどれだけくぐれるか？

という趣旨のゲームになっております。

起動方法：
このリポジトリをダウンロードして、Hawk Eye.exeをクリックしてください。

操作方法：

w s : 上下運動 (up & down)
a d : 旋回運動 (rolling)
右クリック：ブレーキ 
(right click : brake)
リスタート：左上のボタンをクリック 
(restart :  click upperleft button on screen)
